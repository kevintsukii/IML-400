In describing what exactly the W3 stands for, Berners-Lee breaks down the components:

  • Universal Resource Identifiers (addresses for objects and what we now call URLs)
  • Berners-Lee describes HTTP as a "simple request/response" protocol used that passes the string of the address for the server to be contacted and a substring to be passed to the server. The "/" represents a hierarchical space and the "?" is a separator between the address of an object and a query operation applied.
  • Hypertext Markup Language (HTML): a language of communication that "flows" over the network. It is a way for elements to be structured.
